# Comp Sim Audio Sources

All current Comp Sim audio assets in `public/audio/comp-sim` are sourced from Mixkit Free Sound Effects and processed into normalized MP3s for in-app playback. The ambient scene files are now multi-minute composite beds built from the listed Mixkit sources with crossfaded transitions so the live Comp Sim room does not feel like a short clip restarting.

- License reference: https://mixkit.co/license/
- Source host: https://mixkit.co/free-sound-effects/
- Download pattern used: `https://mixkit.co/free-sound-effects/download/<ID>/?context=item+grid`
