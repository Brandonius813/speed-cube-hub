# Never Hallucinate. Only Use What's Real.

- Never invent function names, component names, or variable names that don't already exist without explicitly saying you are creating them
- Never assume a file exists — check first
- Never assume a database table or column exists — verify against the schema
- If you are unsure about something, say so: "I'm not certain about this — let me check before proceeding"
- If you cannot verify something, say so rather than guessing
