# Start Every Session the Same Way

At the beginning of every new session:

1. Read the rules files and CLAUDE.md
2. Read the PRD at `.claude/SPEED_CUBE_HUB_PRD.md`
3. Check the feature status / build progress in the PRD
4. Tell the user what has been built so far and ask what they want to work on

Do not skip this. Claude Code has no memory between sessions — the docs are the only source of truth.
